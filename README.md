10 basic JavaScript logic problems. 

## Problems solved

1.  **Reverse a String**: Reversing input using `split`, `reverse`, and `join`.
2.  **Count Vowels**: Counting 'a, e, i, o, u' in a string using a loop.
3.  **Check for Palindrome**: Validating if a string reads the same forward and backward.
4.  **Find Maximum Number**: Finding the largest value in an array.
5.  **Remove Duplicates**: Using `Set` to filter unique values.
6.  **Sum of Array**: Calculating the total sum of numbers in an array.
7.  **Find Even Numbers**: Filtering even numbers using the modulo operator.
8.  **Capitalize First Letter**: Transforming the first character of each word to uppercase.
9.  **Factorial Calculation**: Using a loop to find the factorial of a number.
10. **PingPong Challenge**: A logic-based loop from 1 to 20 for multiples of 3 and 5.

## How to run

To see the results in your console, simply run:

```bash
node index.js
```
